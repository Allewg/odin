# Imágenes para ODIN GYM

Coloca las siguientes imágenes en este directorio:

## Imágenes Requeridas

### Logo
- `logo.png` - Logo principal de ODIN GYM (recomendado: 200x200px, fondo transparente)

### Hero Section
- `hero.jpg` o `hero.mp4` - Imagen/video de fondo para la sección hero (recomendado: 1920x1080px)

### Galería de Instalaciones
- `gallery-pesas.jpg` - Área de pesas
- `gallery-cardio.jpg` - Zona de cardio
- `gallery-funcional.jpg` - Zona funcional
- `gallery-dj.jpg` - Sistema DJ
- `gallery-vestuarios.jpg` - Vestuarios
- `gallery-estiramiento.jpg` - Área de estiramiento

### Servicios
- `servicio-entrenamiento.jpg` - Entrenamiento personalizado
- `servicio-planes.jpg` - Planes de gimnasio
- `servicio-clases.jpg` - Clases grupales
- `servicio-nutricion.jpg` - Nutricionista
- `servicio-kinesiologia.jpg` - Kinesiología

### Equipo
- `equipo-tiare.jpg` - Foto de Tiare Allende (Embajadora)
- `equipo-entrenadores.jpg` - Equipo de entrenadores

### Testimonios (opcional)
- `testimonio-1.jpg` - Foto de cliente (opcional)
- `testimonio-2.jpg` - Foto de cliente (opcional)
- `testimonio-3.jpg` - Foto de cliente (opcional)

## Especificaciones Técnicas

- **Formato:** JPG, PNG o WebP
- **Tamaño recomendado:** 
  - Hero: 1920x1080px (16:9)
  - Galería: 1200x1200px (1:1) o 1200x800px (3:2)
  - Logo: 200x200px o 400x400px
- **Optimización:** Comprimir imágenes para web (usar herramientas como TinyPNG, ImageOptim)
- **Peso máximo recomendado:** 500KB por imagen

## Notas

- Las imágenes deben ser de alta calidad
- Considera usar imágenes de personas entrenando para mayor autenticidad
- Mantén la consistencia en el estilo visual (colores, iluminación)
- Asegúrate de tener los derechos de uso de todas las imágenes
